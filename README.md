# iKuuu 自动签到脚本
支持青龙面板每日自动签到领取流量，多账户管理，推送通知

## 功能特性
- ✅ 每日自动签到，自动领取流量
- ✅ 签到前获取剩余流量，签到结果完整展示
- ✅ 支持多账户批量签到
- ✅ 支持青龙原生通知 + Pushplus 推送
- ✅ 域名支持通过环境变量配置，换域名无需修改脚本

---

## 青龙面板 环境变量配置
在青龙面板 → 环境变量 页面，添加以下变量：

| 变量名          | 是否必填 | 说明                                                                 |
|-----------------|----------|----------------------------------------------------------------------|
| `IKUUU_COOKIES` | ✅ 必填   | 你的iKuuu账号Cookie，多个账号一行一个，直接填原始Cookie即可 |
| `IKUUU_HOST`    | ⚪ 可选   | iKuuu站点域名，默认 `ikuuu.win`，换域名直接修改变量值就行         |
| `PUSHPLUS_TOKEN`| ⚪ 可选   | Pushplus推送token，留空则仅使用青龙自带通知                         |

### 获取Cookie说明
登录iKuuu网页版 → 打开个人中心 → F12打开开发者工具 → 「网络」标签 → 刷新页面 → 任意点击一个请求 → 在请求头中找到 `Cookie`，复制完整内容填入即可。
你的Cookie必须包含 `uid` `email` `key` 三个参数才能正常使用。
